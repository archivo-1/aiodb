Theme Lab

Editor visual de temas para tus dirArcs en Aio Repo.

Diseña sin tocar código

Theme Lab te deja ajustar colores, fondo, branding y comportamiento visual de thumbnails para cada dirArc, y guarda todo en theme_config del registro correspondiente.

  • Elige un dirArc y cambia su apariencia sin editar archivos manualmente.
  • Trabaja con presets fáciles de entender y un preview instantáneo.
  • Funciona por sí solo con Supabase, así puedes probarlo en Live Server antes de integrarlo al dashboard.

Iniciar sesión

Usa la misma cuenta que ya administra tus dirArcs.

Tus dirArcs

Selecciona el directorio que quieres personalizar.

Configura el tema

Selecciona un dirArc para comenzar.

Aún no hay un dirArc seleccionado. Cuando elijas uno en la columna izquierda, aquí aparecerán los controles visuales.

Preset base

El preset ayuda a partir desde una idea clara. Puedes cambiarlo y luego ajustar cada detalle manualmente.

Colores

Estos colores corresponden a la base visual que ya usa Aio Repo: fondo, superficies, bordes, texto y acento.
Puede ser rgba() para mantener transparencia fina.
Se usa para pills activas, botones suaves y fondos de énfasis.

Fondo

Puedes dejar un color sólido o usar una imagen con overlay para reforzar el carácter visual.
0.18 · mayor valor = más oscurecimiento sobre la imagen.

Branding

Controla el título que se ve en header, subtítulo y logo de cabecera.

Acciones

Guarda el tema en theme_config del dirArc seleccionado. El prototipo guarda todo aunque el consumidor final todavía deba aplicar esos valores visualmente en Aio Repo.
