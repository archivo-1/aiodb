Mis Aio Repo

Selecciona un Aio Repo para gestionar sus contenedores.

    Contenedores

    Ningún Aio Repo seleccionado.
    Inicia sesión y selecciona un Aio Repo en la columna izquierda.

    Nuevo Aio Repo

    Un Aio Repo es un “content.json virtual” que agrupa tus contenedores en colecciones. Luego lo verás en ArchVista con los visores.

    Enlace de Aio Repo: index.html?slugDirArc=…

    Nuevo contenedor

    Cada contenedor corresponde a una URL (archivo o enlace) más metadatos. ArchVista decide el visor según el tipo.

    Activado → enlace (iframe).
    Desactivado → enlaceEXT (tarjeta, nueva pestaña).

    Cambiar contraseña

    Ingresa una nueva contraseña para tu cuenta de Aio Repo.
